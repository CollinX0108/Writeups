# HTB Writeups

Repositorio público con writeups de máquinas públicas de Hack The Box que he resuelto.

## Estructura

```
Writeups/
├── HTB/
│   ├── Oopsie/
│   │   ├── writeup.md
│   │   └── images/
│   └── [Otras máquinas]/
└── README.md
```

## Máquinas Resueltas

- [Oopsie](./HTB/Oopsie/writeup.md)

## Notas

- Estos writeups son de máquinas públicas, por lo que no contienen información sensible
- Los writeups están escritos en formato Markdown y pueden ser convertidos a PDF usando Pandoc
- Cada writeup incluye screenshots y comandos utilizados durante la resolución

## Herramientas Utilizadas

- Nmap
- Burp Suite
- Metasploit
- Otras herramientas según sea necesario

## Agregar una Nueva Máquina

1. Crea un nuevo directorio: `mkdir -p HTB/NombreMaquina/images`
2. Copia el template: `cp template.md HTB/NombreMaquina/writeup.md`
3. Edita el writeup.md con la información de la máquina
4. Agrega screenshots en el directorio `images/`
5. Actualiza el README.md con la nueva máquina
